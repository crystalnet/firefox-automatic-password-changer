Passwörter generieren

Bei Rechtsklick auf ein Eingabefeld kann das Add-On zum Generieren eines Passwortes genutzt werden, indem man auf die Option "Neues Passwort generieren" klickt. Wenn ein neues Passwort generiert wurde, ist bei einem weiteren Rechtsklick auf ein Eingabefeld einmalig die Option "Zuletzt generiertes Passwort verwenden" auswählbar. Auf diese Weise ist es möglich, Passwort-Formulare auszufüllen, die eine erneute Eingabe des Passwortes verlangen.

Die Optionen zum Passwort-Generieren stehen jederzeit zur Verfügung, unabhängig davon, ob gerade eine Blaupause aufgezeichnet wird oder nicht.

Eine Blaupause aufzeichnen

Alle Aktionen während des Aufzeichnens müssen per Klick durchgeführt werden. Ausgenommen davon ist die Eingabe von Text.

  1. Die aufzuzeichnende Seite aufrufen.
  2. Über den Add-On-Button im Menü auf "Aufzeichnen" klicken oder per Rechtsklick im Kontextmenü die Option "Aufzeichnen" wählen.
  3. Zur Loginmaske navigieren. Alternativ kann die Aufzeichnung auch erst auf dieser Seite begonnen werden. Der Login muss allerdings auf jeden Fall Teil der Aufzeichnung sein.
  4. Die Nutzerdaten eingeben und vor dem Absenden per Rechtsklick auf die Eingabefelder diese entsprechend als "Benutzername/Email" bzw. "Aktuelles Passwort" markieren. Sind die Felder korrekt markiert, werden diese gelb hinterlegt.
  5. Zur Passwort-Ändern-Maske navigieren.
  6. Aktuelles und neues Passwort eingeben und vor dem Absenden analog zur Loginmaske die Eingabefelder per Kontextmenü markieren.
  7. Passwort-Richtlinien in dem Fenster eingeben, das beim Markieren des Eingabefeldes für das neue Passwort erscheint.
  8. Passwortänderung abschließen und Ausloggen.
  9. Über den Add-On-Button im Menü oder per Rechtsklick im Kontextmenü die Aufzeichnung beenden.

Accountliste

Die Accountliste wird mit einem Klick auf den Add-On-Menüpunkt "Accountliste" geöffnet.

Der Button rechts oben ermöglicht das Importieren von Blaupausen.

Es gibt drei mögliche Arten von Einträgen auf der Liste:

  1. Es liegen für eine Webseite Logindaten im Firefox-Passwortmanager vor und es ist eine Blaupause vorhanden.
    • Das Vorhandensein einer Blaupause für diese Webseite wird durch ein Icon angezeigt (1).
    • Bei einem Klick auf (2) ändert das Add-On automatisch das Passwort für diese Webseite. Hierbei wird ein automatisch generiertes Passwort genutzt.
    • Ein Klick auf (3) öffnet einen Tab, in dem die Passwort-Ändern-Maske der Webseite aufgerufen wird, damit das Passwort manuell geändert werden kann.
    • Der Button (4) ermöglicht es, für diese Webseite eine neue Blaupause aufzunehmen. Die alte Blaupause wird dabei überschrieben.
    • Löschen oder Exportieren der Blaupause ist mittels (5) möglich.
  2. Im Firefox-Passwortmanager liegen Logindaten für die Webseite vor, es ist jedoch keine passende Blaupause vorhanden.
    • Ein Klick auf (1) startet die Aufnahme einer Blaupause für diese Webseite.
  3. Es liegt eine Blaupause für eine Webseite vor, für die keine Logindaten im Firefox-Passwortmanager vorhanden sind. Dies kann z.B. der Fall sein, wenn eine Blaupause importiert wurde.
    • Ein Icon zeigt an, dass für diese Webseite eine Blaupause vorliegt (1).
    • Mittels (2) kann die Blaupause gelöscht oder exportiert werden.
