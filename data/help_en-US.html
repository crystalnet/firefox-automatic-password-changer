Hilfeseite

Einen Blueprint aufzeichnen

  • Addonbutton auswählen
  • Aufzeichnen drücken
  • Die aufzuzeichnende Seite aufrufen.
  • Bis zur Loginmaske klicken
  • Rechtsklick auf das Feld der Emailadresse/Benutzername
  • Markieren als/Emailadresse/Benutzername
  • Rechtsklick auf das Feld des Passwortes
  • Entweder Passwort einfügen und geeignetes Passwort auswählen. oder
  • Markieren als aktuelles Passwort
  • Über Klicks zur Passwort-Ändern-Maske navigieren
  • Das Feld des aktuellen Passwortes:Rechtsklick/Markieren als/aktuelles Passwort
  • Die Felder des neuen Passwortes: Rechtsklick/Markieren als/neues Passwort
  • Dann die Eingaben von Hand tätigen
  • Passwortänderung abschließen
  • Ausloggen
  • Über den Addonbutton auf dem Optionsfeld die Aufzeichnung beenden

Das Passwort ändern

  • Addonbutton auswählen
  • Accountliste drücken
  • Auf den Passworteintrag der entsprechenden Seite klicken
  • (Es muss ein Blueprint für die Seite existieren. Ansonsten weiter unter -> "Blueprints verwalten")
  • Jetzt kann man "Passwort automatisch ändern" oder "Passwort jetzt manuell ändern" auswählen
  • Bei "Passwort jetzt manuell ändern" wird man auf die Seite geleitet, auf der man das Passwort manuell ändern kann
  • Bei "Passwort automatisch ändern" wird das Passwort automatisch geändert

Blueprints verwalten

  • Addonbutton auswählen
  • Accountliste drücken
  • Hier wird nun eine Übersicht der Passwortdatenbank angezeigt
  • Einträge mit diesem Symbol besitzen einen Blueprint:
  • Diese können gelöscht
  • Oder auf die Festplatte exportiert werden
  • Alle anderen Einträge lassen sich aufklappen
  • Nach dem Aufklappen kann mit dem Button "Blaupause aufzeichnen" die noch nicht vorhandene Blaupause aufgezeichnet werden
